【自動排班工具】

排班的本質是溝通與公平。透過智慧化演算法，【自動排班工具】能全面取代繁瑣的人工計算，實現排班自動化與完美格式輸出。

核心功能一覽：
1. 智慧出勤判斷與分組
排休保護機制：自動剔除請假日（休、特、婚、年等），僅針對當日上班人員進行任務指派。

權限分組管理：支援將幹部組與客服組區隔，各自獨立分配對應業務。

2. 多重業務公平輪替
全自動次數平衡：針對喵喵客服、交接值日生、打掃值日生、印度業務及不同組別的直屬飛機，進行整月頻率統計，確保每位同仁獲派次數趨近一致。

主管階層固定指派：針對主任、大主任等特定職位，支援指定業務固定排班。

3. 模組化與視覺優化
自動曆法適應：自動判讀大小月天數，無需手動微調表格結構。

視覺格式同步：一鍵套用預設的字體、文字顏色與背景色塊，產出具備高可讀性的排班表。
